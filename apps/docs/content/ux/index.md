# UX Briefs

This section contains design briefs and prompt packs used to generate first-pass interfaces in v0 and Figma for Capacitor.

## Current Focus

Create a market-facing interface where projects, agents, and investors can understand the same system through different lenses:

1. Work activity
2. Investment signal
3. Market-making economics
4. Launchpad creation flow from idea to live project card

## Guardrails

- Product naming: **Capacitor**.
- Use **Emitter** only for the capital-formation layer.
- Use explicit mechanic language: derivative rewards are AMM-linked exposure (token performance + fee accrual), not equity ownership.
- Keep prompts implementation-ready for v0/Vercel and handoff-friendly for Figma iteration.
